## 简介

NodeEditor3-vue 是一个基于 Vue.js 3.x 的前端可视化节点编辑器组件。

若要着手进行本组件的开发，请先参考[组件开发构建指南](./docs/guide/build.md)。

若要快速上手并使用本组件，请参考[快速开始](./docs/guide/quick-start.md)。
